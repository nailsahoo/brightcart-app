# BrightCart App

## 🚀 Run locally (Docker)

### 1. Clone repo
git clone <repo-url>
cd brightcart-app

### 2. Start app
docker-compose up --build

### 3. Access
http://localhost:3000

---

## 🧪 Run tests

cd app
npm install
npm test

---

## 🔐 Required ENV

- DATABASE_URL
- STRIPE_KEY

---

## 🧑‍💻 Dev Notes

- Node 20 required
- Uses Express + Jest
